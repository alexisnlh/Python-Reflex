Página web creada utilizando Python y la librería Reflex. La página web consiste en un chat sencillo con OpenAI.

### Estructura del código
> *Dentro del código se puede encontrar comentarios para cada línea o proceso que es importante mencionar su utilidad.*

El proyecto consta de un directorio /.web donde se almacen los archivos de JS compilados (este fichero es creado al inicializar reflex en el proyecto personal), el directorio /assets donde se encuentran los archivos estáticos (imágenes, fuentes, etc.) y en el directorio /curso_reflex es donde se encuentra la lógica de la aplicación.

> *Dentro del directorio /curso_reflex se debe crear un archivo llamado openai_env.py donde se encuentre una variable llamada API_KEY que corresponde a la key personal de OpenAI*
